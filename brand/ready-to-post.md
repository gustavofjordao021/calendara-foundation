# Ready to Post — IG launch sequence (from Thu Jun 18)

Open the folder, upload in numbered order, paste the caption. Files live in Drive `_library/06_social/ready-to-post/`. Captions follow voice rules: no "snap", no exclamation points unless earned. Hashtags can go in the caption or as a first comment (first-comment keeps the caption clean — either works).

**Posting mechanics:** IG carousels post slides in filename order (01→08). Best window for this ICP is ~7–9am or ~8–10pm local (parents before/after the day). Reply to every comment in the first hour — small accounts live on it.

### One-time setup: the link (do this before posting)
Instagram has **no tappable links in feed posts/carousels** — the painted "App Store" button is visual only. The link works in two places:
1. **Bio link** → set it to `usecalendara.com/ig` (redirects to the App Store + tags the install `instagram_bio`). Live on next web deploy.
2. **Stories link sticker** → the one tappable link on organic content. Reshare each new post to your Story with a link sticker to `usecalendara.com/ig`.

So every caption ends with **"Link in bio 👇"** — that's the whole conversion path. (TikTok bio = `usecalendara.com/tiktok`.)
